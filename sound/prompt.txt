Взять из проекта @formulas-audio-lab/index.html @formulas-audio-lab/README.md  код осциллографа и спектрограф звука и выделить как отдельный проект:
- на экране отображаются следующие элементы:
  - кнопки управления: пуск/стоп, запись, настройки 
  - три графика, которые можно свернуть по отдельности (по умолчанию развёрнуты):
    - осциллограф 
    - спектрограмма
    - кепстр https://ru.wikipedia.org/wiki/%D0%9A%D0%B5%D0%BF%D1%81%D1%82%D1%80
- блок настроек позволяет указать частоту дискретизации, битрейт и другие параметры захвата звука (надо динамически определять, какие параметры устройство может захватывать). С этими же настройками будет записываться будущий WAV, если нажмут кнопку записи.    
- у осциллографа должны быть следующие элементы управления: масштаб по Y, частота дискретизации по t
- у спектра и кепстра должны быть следующие режимы: в виде графика (x - частота, y - интенсивность) и спектрограмма (как сейчас по умолчанию). В режиме спектрограммы должны быть следующие элементы управления: масштаб по Y (частоты), линейный/логарифмический режим по Y, скорость обновления (медленно, средне, быстро)
- взять готовый код, чтобы устройство не засыпало во время работы
- запись должна идти в WAV (переиспользовать готовый код)
- при нажатии на запись начинается захват звука (не надо отдельно нажимать старт)
- состояние открыто/скрыто для графиков (только для графиков) сохраняется в local storage, при загрузке страницы восстанавливается прежнее состояние
- весь интерфейс и документация должны вестись на английском языке

Сделанные изменения в структуре и алгоритмах надо отобразить в @sound/README.md  если необходимо. Если есть вопросы или что-то неясно и двойственно, спрашивай меня.
